## Clipboard Manager Daemon

Менеджер буфера обмена с клиент-серверной архитектурой, шифрованием и разграничением прав доступа для Linux.

## Возможности

- Сохранение истории буфера обмена X11
- Шифрование данных алгоритмом AES-256-GCM
- Изоляция данных между пользователями
- Административный доступ ко всем записям
- Мониторинг CLIPBOARD и PRIMARY selections

## Сборка
Необходимо перейти в папку проекта: 
cd ~/clipboard-manager

и собрать проект командой:
make

## Запуск 
# Демон (от root)
sudo ./clipboard_daemon

#Или в фоновом режиме:
sudo ./clipboard_daemon -d

#Мониторинг буфера обмена (в другом терминале)
./monitor.sh

##Опции
#Просмотр истории:
./clipboard_client history

#Административный просмотр (от root)
sudo ./clipboard_client admin-view
